# ISIC 2024 - Выявление рака кожи с помощью 3D-TBP
## Описание
Данный пет-проект является исследовательской работой по выявлению рака кожи с помощью двух разных подходов: 
+ использование табличных данных (мета-данных) и обучение ML-моделей
+ использование снимков и обучение нейронных сетей
### Проблематика
Рак кожи может быть смертельным, если его не выявить на ранней стадии, но многим людям не хватает специализированной дерматологической помощи. За последние несколько лет было доказано, что алгоритмы искусственного интеллекта, основанные на дерматоскопии, помогают клиницистам в диагностике меланомы, базально-клеточного и плоскоклеточного рака. Однако определение того, кому из пациентов следует обратиться к врачу в первую очередь, имеет большое потенциальное значение. Сортировочные системы обладают значительным потенциалом для оказания помощи группам населения, не получающим достаточного обслуживания. На снимках, полученных с помощью дерматоскопа, видны морфологические особенности, невидимые невооруженным глазом, но такие снимки обычно получают только в дерматологических клиниках. Алгоритмы, которые помогают людям в первичной медицинской помощи или в неклинических учреждениях, должны быть приспособлены для оценки изображений низкого качества. Именно поэтому для сбора исследуемых данных исползовалась технология 3D TBP, собранный с огромного количества пациентов с трех разных континентов с изображениями, близкими по качеству с фотографиями, сделанными с мобильного телефона.

### Данные
Данные для исследования были взяты с хакатона [ISIC 2024 - Skin Cancer Detection with 3D-TBP](https://www.kaggle.com/competitions/isic-2024-challenge/overview). Туда входят метаданные и сами снимки. Более подробное описание данных вы можете найти на странице хакатона.
